# gitlab-rancher-kubernetes

## Purpose
How to enable Auto DevOps GitLab CI/CD to a custom Rancher managed Kubernetes cluster

## Tested versions
- VM Ubuntu : 18.04
- Docker : 18.06
- GitLab : 11.9.6-ce
- Rancher : 2.1.8

## References
- Official GitLab documentation : https://docs.gitlab.com/ce/user/project/clusters/
